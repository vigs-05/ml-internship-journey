
**Python OOPs Mastery (Week 1)**

**Goal:** Build a rock-solid foundation in Python Object-Oriented Programming for my ML Internship journey.

This folder contains my daily progress and exercises from Week 1. I focused purely on understanding Classes, Objects, and core OOPs concepts without relying on external libraries.

---

##  Concepts Covered
- Classes and Objects (`__init__`, `self`)
- Instance Variables vs. Class Variables
- Instance Methods, Class Methods (`@classmethod`), and Static Methods (`@staticmethod`)
- Using Data Structures inside Objects (Lists `[]` and Dictionaries `{}`)
- Object Interaction (Multiple classes communicating with each other)

---

##  Project Breakdown

| File Name | Concept Learned | Description |
| :--- | :--- | :--- |
| `car.py` | Attributes | Practicing default attributes and string formatting. |
| `bank_account.py` | Logic | Implementing `if/else` conditions inside methods to prevent overdrafts. |
| `student_grades.py` | Lists in OOPs | Storing multiple data points (grades) in a list inside an object. |
| `employee_tracker.py` | Class Variables | Modifying a shared variable (`total_employees`) across all objects. |
| `calculator.py` | Decorators | Self-taught `@staticmethod` and `@classmethod` for utility functions. |
| `shopping_cart.py` | Dictionaries in OOPs | Adding/removing items from a dictionary and calculating totals. |
| `library_management.py` | **Week 1 Boss** | Multi-class interaction. A `Library` managing `Book` and `Member` objects. |

## How to Run
1. Make sure Python 3.x is installed on your system.
2. Clone this repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/ml-internship-journey.git
   ```
3. Navigate to the week 1 folder:
   ```bash
   cd ml-internship-journey/week_1
   ```
4. Run any file:
   ```bash
   python library_management.py
   ```
